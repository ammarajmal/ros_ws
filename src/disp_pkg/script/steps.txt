1. Image/Frame read (camera1/image_raw)
2. Camera Parameters read (camera1/camera_info)
3. ARUCO DICT initialize (declare)
4. Detect ARUCO Marker using (detectMarkers() function) - it gives 2 important parameters (1. corners, 2. ids)
5. Estimate Pose using either of these functions
    estimatePoseSingleMarkers()
    or solvePnP()
    -> it gives tvecs[0], and rvecs[0].
6. convert rotation vector (rvec) to rotation matrix using (Rodrigues = cv2.Rodrigues() function))
7. take inverse of Rotation matrix
8. Get 3d displacement in xyz as => 3D_Displacement =  -R[prev] x inverse(R[cur]) (T(cur)-T(prev))
